# Ярлык для Яндекс Читалки

Позволяет создать независимый ярлык быстрого доступа на рабочем столе телефона для мгновенного запуска встроенной Читалки (Библиотеки) внутри приложения Яндекс.

## Зачем это нужно?

В экосистеме мобильного приложения Яндекс есть удобная читалка электронных книг. Однако отдельной иконки или быстрого ярлыка для неё Яндекс не предоставляет.

Этот сайт позволяет создать на рабочем столе телефона ярлык Читалки, который Яндекс в свое приложение не встроили.

---

Создание ярлыка через этот сайт работает только в приложении [Яндекс (Яндекс — с Алисой)](https://yandex.ru/all?mode=apps&service=searchapp).

Не работает с приложением Яндекс Браузер.
